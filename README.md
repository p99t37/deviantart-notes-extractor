## Tips for good screenshots

- Use a clean browser window (no extra tabs/extensions visible)
- Crop tightly to the relevant area
- PNG format, reasonable resolution (1x or 2x)
- If you want the banner, you can make one at [canva.com](https://canva.com) or similar
